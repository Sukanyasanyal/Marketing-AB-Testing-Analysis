Statistical A/B testing in Python to optimize marketing conversion rates
# Marketing Campaign A/B Test Analysis
This project uses **Python** and **Inferential Statistics** to evaluate the success of two marketing strategies.

## Key Accomplishments:
* **Statistical Testing:** Performed a Z-test for proportions to validate campaign performance.
* **Business Results:** Identified a **9.1% lift** in conversion rates ($P < 0.001$).
* **Tools Used:** Python (Pandas, Statsmodels, Matplotlib), Jupyter Notebook.
